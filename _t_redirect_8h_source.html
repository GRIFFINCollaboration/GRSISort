15/// A simple class to redirect stdout and stderr to file(s). To
16/// use it, create it, providing the path of the files stdout and
17/// stderr should be redirected to, and an (optional) flag whether
18/// to append to those files (default) or not.
19/// If only one file is provided, both stdout and stderr are
20/// redirected to it.
