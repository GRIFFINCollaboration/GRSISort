TILLDataParser Class Reference
Sorting

Detailed Description

The TILLDataParser is the DAQ dependent part of GRSISort. It takes a "DAQ-dependent"-flavoured LST file and converts it into a generic TFragment that the rest of GRSISort can deal with. This is where event word masks are applied, and any changes to the event format must be implemented.
